

This is the Readme for downloading and installing the Tangible Scores app compiled for Mac OSX. It is only prepared for running under OSX. It can be adapted for other operating systems.  

1) Download it from www.ultranoise.es/tangiblescores/TangibleScores5.zip
2) Extract somewhere in your computer
3) Download the externals.zip file you should see in this repository folder
4) Extract and copy them to "/YOUR-COMPUTER-NAME-AND USER/Documents/Pd/externals"
5) Run the app Tangible Scores 5 to open the GUI control program. 
6) Pure Data 0.48-1 will also be opened. Configure Pd Audio Settings for your system.
